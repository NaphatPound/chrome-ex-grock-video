# Grok AI Video Generator Chrome Extension

Auto-upload images to Grok AI and generate videos with custom prompts. Supports loop generation and automatic downloads.

## Features

- **Image Upload**: Select multiple images to process
- **Custom Prompts**: Set your own video generation prompts
- **Loop Generation**: Generate multiple videos per image
- **Continuous Mode**: Keep generating videos indefinitely
- **Auto-Download**: Automatically download completed videos
- **Progress Tracking**: Monitor generation progress in real-time
- **Logging**: View detailed logs of all operations

## Installation

1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode" (toggle in top-right corner)
3. Click "Load unpacked"
4. Select this extension folder
5. The extension icon will appear in your toolbar

## Usage

1. Click the extension icon to open the popup
2. Enter your video generation prompt
3. Select one or more images to upload
4. Configure options:
   - **Loop Count**: Number of videos to generate per image
   - **Delay Between Generations**: Wait time between generations (in seconds)
   - **Auto-download**: Automatically save completed videos
   - **Continuous Mode**: Keep generating indefinitely
5. Click "Start Generation"
6. The extension will open Grok and begin generating videos

## Notes

- You must be logged into Grok at grok.com
- Video generation may take several minutes per video
- Downloaded videos are saved to your default downloads folder
- The extension works with `grok.com/imagine`

## Troubleshooting

- If the extension can't find the input area, try refreshing the Grok page
- Make sure you're logged into X (Twitter)
- Check the extension log for error messages
- If videos aren't downloading, check your browser's download settings
